# Compare Simple second with Common Time

Read more about the project here: https://sssecond.com

## Requirements

* Python 3
* Node.js

## Interface

```
python3 main.py -h
usage: main.py [-h] step {W,D,h,m,s} spread

https://sssecond.com

positional arguments:
  step         Step size
  {W,D,h,m,s}  Step unit (weeks, days, hours, minutes, seconds)
  spread       Number of steps

optional arguments:
  -h, --help   show this help message and exit
```
